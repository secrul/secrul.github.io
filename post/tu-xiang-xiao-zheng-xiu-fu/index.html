图像校正、修复、填充

2020-08-07

几何失真：在许多实际图像采集处理过程中，图像中像素之间的空间关系会发生变化，简单说就是直线不直，圆形不圆，称之为几何失真或者几何畸变。


原始场景中各个像素的位置关系和图片中像素的空间关系不一致。需要通过对坐标的几何变换来校正各个像素的位置，恢复原有的空间关系。几何失真校正工作包括：

  1. 空间变换：对图像平面上的像素进行重新排序，恢复原空间关系
  2. 灰度插值：对空间变换后的像素赋予相应的灰度值来恢复原位置的灰度。
    ##1.空间变换
    原图像f(x,y)，受到几何形变后变成了g(x′,y′)g(x^{'},y^{'})g(x′,y′).这里(x′,y′)(x^{'}, y^{'})(x′,y′)表示失真图像的坐标。失真过程可以表示为：
    {x′=s(x,y)y′=t(x,y)\begin{cases} x^{'} = s(x, y) \\y^{'} = t(x, y)\end{cases}{x′=s(x,y)y′=t(x,y)​
    s(x, y)和t(x, y)分别代表着两个空间变换函数。当是线性失真的时候：
    {s(x,y)=k1∗x+k2∗y+k3t(x,y)=k4∗x+k5∗y+k6\begin{cases} s(x, y) = k_{1}*x + k_{2}*y + k_{3} \\t(x, y) = k_{4}*x + k_{5}*y + k_{6}\end{cases}{s(x,y)=k1​∗x+k2​∗y+k3​t(x,y)=k4​∗x+k5​∗y+k6​​
    而对于一般的二次失真(非线性)：
    {s(x,y)=k1∗x2+k2∗y2+k3∗x∗y+k4∗x+k5∗y+k6t(x,y)=k7∗x2+k8∗y2+k9∗x∗y+k10∗x+k11∗y+k12\begin{cases} s(x, y) = k_{1}*x^{2} + k_{2}*y_{2} + k_{3}*x*y + k_{4}*x +k_{5}*y + k{6} \\t(x, y) = k_{7}*x^{2} + k_{8}*y^{2} + k_{9}*x*y + k_{10}*x + k_{11} * y + k_{12}\end{cases}{s(x,y)=k1​∗x2+k2​∗y2​+k3​∗x∗y+k4​∗x+k5​∗y+k6t(x,y)=k7​∗x2+k8​∗y2+k9​∗x∗y+k10​∗x+k11​∗y+k12​​
    **关键是找到失真函数。**实际中不知道失真函数表达式，需要在失真图和校正图上找到一些确切的点(约束对应点)，利用这些点计算出失真函数。
    ##2.灰度插值
    在进行空间变换的时候两幅图像素坐标可能不会同时是整数，有两种角度分析，一是失真图像中整数坐标的像素经过变换后坐标不再是整数(前向映射)，另一是变换后图片的整点坐标对应着原失真函数的非整点坐标(后向映射)。
    对于前向映射：当失真图像是整数坐标，复原图像不是整数坐标时，将失真图像像素的灰度值按照距离分配给复原图像中临近的四个像素点。
    对于后向映射：复原图像中的整数坐标对应着失真图像中的非整数坐标，将失真图像中四个像素点的灰度值加权平均后赋给复原图像的整数点坐标。

    前向映射缺点：
  3. 失真图像的一些像素点会被映射到复原图像之外，计算浪费
  4. 一个复原图像的像素点对应着许多失真图像里的像素点，需要更多的寻址操作，特别是在高阶插值中。
  5. 不能采用最邻近插值
  6. 复原图像中的一些像素在失真图像中找不到对应点
    后向映射应用更为广泛
    ###灰度插值的计算
  7. 最近邻插值：(零阶插值)，将距离(x′,y′)(x^{'},y^{'})(x′,y′)最近像素点的灰度值赋给复原图中的(x,y)。计算量小，但是不够精确。
  8. 双线性插值：选定复原图像中一个像素点所对应的失真图像中4个像素点，假定灰度值在x方向和y方向线性分布，计算出$(https://secrul.github.io/post-images/1596857507026.jpg)
  9. 三次线性插值：将双线性插值推广到三维。三次指利用线性插值用了三次。


    #图像修复
    在图像采集、传输、处理过程中，可能发生图像部分区域缺损，像素灰度发生急剧变化使图像不完整。
    图像修补是要基于不完整的图像和对原始图像的先验知识，通过采用相应的方法纠正或校正区域缺损问题。修补可以分为修复和补全。前者一般指小范围的插补，后者指区域填充。
    图像修补的困难来源于三个方面：
  10. 领域复杂性：需要修补的区域随应用领域而不同，文字区块是字符，图像中可能是任意形状的。
  11. 图像复杂性：图像性质在不同的尺度有着不同的表现形式，小尺度有很多细节，而大尺度可以用平滑函数
  12. 模式复杂性：必须考虑有意义的高层模式。
    全变分模型：
    应用于划痕或者尺度较小的靶区域的技术，大都基于偏微分方程或者变分模型，两者可以借助变分原理相互等价推出。
    需要先看泛函数...
    混合模型
    全变分模型中扩散只向梯度的正交方向进行。当扩散在区域轮廓附近进行时，可以很好的保持边缘，但是在区域的内部，边缘方向会很随机，此时受噪声影响可能产生虚假轮廓。
    需要先看全变分
    #区域填充
    对于缺失区域较大图像不适合用修复的方法，一方面扩散会造成图像模糊，另一方面区域内外纹理特性可能有较大的差别。
    区域填充思路：
  13. 将图像分解为结构部分和纹理部分，对结构性强的部分仍可用扩散的方法，而对纹理明显的部分可以借助纹理合成技术进行填充。
  14. 在图像未退化部分选择一些样本块，依据这些样本块逐步对区域内部填充。
    基于样本块的方法
    基于样本的图像补全方法使用保持原状的空间区域去估计和填充缺失信息。为此对每个靶区域的像素赋予一个灰度值(0表示尚未填充)和一个置信度。在填充过程中，对除余填充锋线上的图像块赋给一个临时的优先值，以此确定被填充块的填充顺序，整个过程是迭代的。
  15. 计算图像块的优先权：考虑到保持结构信息的需要，这个值在连续边缘上以及被高置信度像素围绕的图像块会比较大。换句话说，先填充具有较强连续边缘的区域和其中已知信息较多的区域。
    对于以边界点p为中心的区块优先值如下：P(p)=C(p)∗D(p)P(p) = C(p) * D(p)P(p)=C(p)∗D(p)C(p)表示置信度，是当前区块中完好像素所占比例，D(p)是数据项，是等照度线与单位法向量的内积。使得沿着等照度线的方向修复，可以很好保持结构信息。
  16. 传播纹理和结构信息：根据计算出的优先权值，选出权值最高的待填充图像块，然后从源区域选出图像块填充它。在选择源图像块的时候，要使两个图像块中已知像素的平方差最小。
  17. 更新置信度：填充完毕后更新置信度C(p)
