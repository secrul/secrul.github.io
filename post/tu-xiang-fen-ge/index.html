图像分割

2020-08-10
  1. 分割出来的各个区域对某些特征(灰度、颜色、纹理)而言具有相似性，区域内连通没有很多小孔
  2. 相邻区域对分割所依据的性质存在明显差异
  3. 区域边界明确

1.边缘检测

基于一阶导数的边缘检测算子有Roberts、Sobel、Prewitt。拉普拉斯算子是基于二阶导数，并且对噪声敏感，一般先对图像进行平滑滤波处理。Canny算子并不是基于微分方法。

1.1梯度算子

要想检测图像边缘，即灰度值变换剧烈的区域，自然想到求导，即求梯度。但是图像是离散的，只好用差分来代替微分。

fx(x,y)=f(x+1,y)−f(x,y)fy(x,y)=f(x,y+1)−f(x,y)f_x(x,y) = f(x+1,y) - f(x,y)\\f_y(x,y) = f(x,y+1) - f(x,y)fx​(x,y)=f(x+1,y)−f(x,y)fy​(x,y)=f(x,y+1)−f(x,y)

(x,y)处的梯度也可以如下方式来近似：

fx(x,y)=f(x+1,y)−f(x−1,y)fy(x,y)=f(x,y+1)−f(x,y−1)f_x(x,y) = f(x+1,y) - f(x-1,y)\\f_y(x,y) = f(x,y+1) - f(x,y-1)fx​(x,y)=f(x+1,y)−f(x−1,y)fy​(x,y)=f(x,y+1)−f(x,y−1)

此处不用关心系数的问题，求得每个点的梯度后将所有像素点的梯度映射到[0,1]区间

1.2Canny算子

基于最优化思想的边缘检测算子，试图在抑制噪声和准确定位之间寻找一个最佳的折中。步骤:

  1. 高斯滤波器对图像进行平滑
  2. 计算平滑后图像的梯度的幅度和方向
  3. 对梯度幅度应用非极大值抑制，找出图像梯度中的局部极大值，把其他非局部极大值的点置为0，得到细化边缘
  4. 用双阈值算法检测和连接边缘，阈值T1，T2(T1>T2),T1用来找到每条线段，T2在这些线段上延申，寻找边缘的断裂处并连接。

2.边界跟踪

从一个边缘点出发，依据一定的贵族搜索下一个边缘点，从而跟踪出目标边界。分为三步：

  1. 确定起始搜索点，不同的起始点结果可能不同
  2. 确定合适的边界判别规则和搜索规则，判别规则是判断一个点是不是边界点，搜索规则是如何搜索下一个点
  3. 确定搜索的终止条件

霍夫变换：将检测到的边缘点连接起来得到边缘曲线，实际上是利用分散的边缘点进行曲线逼近，也可以看成一种聚类分析技术。

y=ax+b  ⟺  b=−xa+yy = a x + b \iff b = -xa + yy=ax+b⟺b=−xa+y

步骤：

  1. 建立一个二维数组A(a,b)，存储任意两点形成直线的斜率a和截距b，初始值为0
  2. 利用霍夫变换计算a，b的值，A(a,b)加一
  3. 选取最大的A(a,b)，对应的a，b就是直线的斜率和截距

3.阈值分割

通过设置阈值将像素点按照灰度值分成若干类，从而实现图像分割。在二值化中，阈值通常利用灰度直方图求双峰，选择谷底为阈值。

阈值选择方法：

  1. 人工选择法：利用先验知识，人工选择
  2. 自动阈值法：1迭代式 2Otsu阈值选择(类间方差最大原则) 3最小误差阈值选择 4分水岭法

4.区域分割

阈值分割法没有考虑空间关系。区域分割利用空间特性，认为分割出来属于同一类的像素点具有相似性质。该方法是一种迭代方法，空间时间开销大

  1. 区域生长法：1选择若干种子点.2确定相似性度量(平均灰度值、纹理、颜色)3.确定生长的终止条件
  2. 区域分割法：1.确定分裂准则 2确定分裂方法
  3. 区域合并法：1.初始区域分割 2相邻区域是否满足满足一致性准则，若满足则合并 3重复合并，直至不能合并
  4. 区域分裂合并法：当满足分裂条件就分割，满足合并条件就合并
